
# Snake: Ultimate Skins Edition

Versión moderna del clásico Snake con tienda integrada, skins, DLCs y paquetes de monedas.

## Características
- Jugabilidad retro con mejoras modernas
- Tienda integrada (monedas, skins)
- Packs DLC (canjeables desde el juego)
- Skins (preview en tienda)
- Guardado en JSON
- Build automático con GitHub Actions (Windows & Linux)
- Sistema de auto-actualización opcional
